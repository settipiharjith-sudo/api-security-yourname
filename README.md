1. Consequences of exposing API key:

Exposing an API key on GitHub allows attackers to misuse it, leading to unauthorized requests, financial charges, and service abuse. It can also result in account suspension and compromise of application security.

2. Why privacy policy prohibits logging city names:

City names are considered personal data and can reveal user location. Logging them violates privacy principles like data minimization and user consent under regulations such as GDPR, risking legal and ethical issues.
